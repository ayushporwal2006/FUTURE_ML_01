# FUTURE_ML_01
Sales Forecasting using Machine Learning
🚀 Overview

This project builds a regression-based sales forecasting system to predict future demand using historical sales data. It helps businesses make better decisions in inventory, staffing, and financial planning.

🎯 Objective

To forecast future sales by learning patterns from past data using lag-based regression techniques.

🛠️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
Matplotlib
🔧 Workflow
Data cleaning & preprocessing
Time-based feature engineering
Lag feature creation (lag_1, lag_7)
Regression model training
Model evaluation (MAE, RMSE)
Visualization of predictions
📊 Visualization Highlights
Total Sales, Profit, Discount & Quantity KPIs
Actual vs Predicted Sales Forecast
Correlation Heatmap
Sales by Region
Sales Trend by Year
Discount Impact on Profit
📈 Key Insights
Sales follow time-based patterns
Discounts increase sales but reduce profit margins
Past sales strongly influence future demand
Profitability depends on pricing strategy
💼 Business Value
📦 Better inventory planning
💰 Improved financial forecasting
👥 Efficient staffing decisions
📊 Data-driven decision making
📌 Conclusion

A regression model with lag features can effectively forecast sales and support real-world business planning.
